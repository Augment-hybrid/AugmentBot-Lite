## Setup

1. Clone repo
2. Add `.env` file with:
   ```
   DISCORD_BOT_TOKEN=your_token_here
   OPENAI_API_KEY=your_api_key_here
   ```
3. Run:
   ```
   pip install -r requirements.txt
   python bot.py
   ```
